1. Get the count of favorites for a given movie:

      
    GET /likes/{movie-id}
The response body will contain the current number of likes for the given movie.

2. Increase or decrease the count of favorites for a given movie:

      
    POST /likes/{movie-id}?change=+1
    POST /likes/{movie-id}?change=-1
The response body will contain the updated number of likes for the given movie.
